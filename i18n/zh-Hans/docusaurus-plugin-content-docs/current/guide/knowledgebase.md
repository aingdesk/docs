# 知识库
## 新建知识库
1. 点击左侧导航栏中的`知识库-新建知识库`。
2. 输入知识库名称，选择嵌入模型，点击`确定`。
![新建知识库](img/createknowledgebase.png)
3. 点击`上传文件`，选择本地文件。
![上传文件](img/uploadkonow.png)
4. 点击确定上传。
![确认上传](img/addfileknow.png)
5. 上传完成后等待知识库嵌入。
![生成知识库](img/emfile.png)

## 使用知识库
1. 在聊天界面，选择`知识库`。
2. 输入问题，点击`发送`。
![使用知识库](img/useknow.png)
